## Git Lessons
- `git reset --soft origin/main` is destructive — it reset all uncommitted queue changes. Never use it after building on top of main.
- Force push to main is risky on team repos; on solo repos it's OK but loses reflog context.
- Prefer: branch → PR → merge workflow for all code changes.
